# H18EQ01

## Standards

### Code

**Nom de Classes**   : UpperCamelCase
**Nom de Méthodes**  : UpperCamelCase
**Nom de variables** : lowerCamelCase

Le code doit être bien indenté.
Les parenthèses "{}" doivent être come ceci : 
```
void NomMethode()
{
    
}
```

### Répertoires

**Ressources**
